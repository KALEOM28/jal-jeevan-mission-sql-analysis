# 🚰 Jal Jeevan Mission SQL Data Analysis Project

## 📌 Project Overview

This project focuses on analyzing Jal Jeevan Mission (Har Ghar Jal) data using SQL and MySQL. The objective is to evaluate district-level performance, identify data quality issues, calculate statistical measures, and generate meaningful insights that support decision-making.

The analysis includes data validation, cleaning, aggregation, ranking, and performance evaluation using advanced SQL techniques.

---

## 🎯 Objectives

* Analyze district-wise performance data
* Identify missing and inconsistent records
* Evaluate water supply coverage metrics
* Calculate statistical measures for performance comparison
* Generate actionable insights through SQL queries

---

## 📊 Analysis Performed

### Data Validation

* Missing value identification
* Duplicate district detection
* Data consistency checks

### Statistical Analysis

* Total record count
* Average performance score
* Maximum and minimum performance
* Variance calculation
* Standard deviation calculation
* Coefficient of variation

### District-Level Analysis

* Top performing districts
* Low performing districts
* Ranking analysis
* Gap identification

### Data Cleaning

* TRIM function usage
* COALESCE function implementation
* Null value handling

### Advanced SQL Operations

* Aggregate Functions
* GROUP BY
* ORDER BY
* HAVING Clause
* Subqueries
* Window Functions
* ROW_NUMBER()
